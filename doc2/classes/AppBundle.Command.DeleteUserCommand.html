\AppBundle\Command

\AppBundle\CommandDeleteUserCommand

A command console that deletes users from the database.

To use this command, open a terminal window, enter into your project directory and execute the following:

$ php bin/console app:delete-user

Check out the code of the src/AppBundle/Command/AddUserCommand.php file for the full explanation about Symfony commands.

See http://symfony.com/doc/current/cookbook/console/console_command.html
